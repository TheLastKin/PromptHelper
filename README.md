An Electron app to start hoarding prompt.
How to test on your PC:
1. Clone the repo: git clone https://github.com/TheLastKin/PromptHelper.git
2. Install NodeJS/NPM
3. Run npm install in the terminal
4. Test the project with npx electron-forge start
5. Package the project with npx electron-forge make. Your app will be at ~rootproject/out/...

"How do I know if this isn't just another virus/malware?" Well, you can have ChatGPT check the whole repo. Or just know that Windows Defender hasn't reacted yet.
